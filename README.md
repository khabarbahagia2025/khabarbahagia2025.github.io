# Undangan Pernikahan Wahyu & Magfi

Undangan pernikahan digital — satu file HTML statis (Tailwind CDN), di-host via GitHub Pages.

**Live:** https://khabarbahagia2025.github.io

## Query parameter

| Parameter | Contoh | Fungsi |
|-----------|--------|--------|
| `to` | `?to=Budi` | Nama tamu di cover |
| `event` | `?event=resepsi` / `?event=ngunduh` | Tampilkan hanya acara tersebut (default: keduanya) |

Contoh: `https://khabarbahagia2025.github.io/?to=Budi&event=ngunduh`

## Struktur

- `index.html` — seluruh halaman (markup, style, script)
- `img/` — foto, favicon, logo bank
- `audio/` — musik latar
